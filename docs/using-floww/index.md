# Using Floww

Learn how to build trigger-based workflows with Floww SDK.

## What You'll Learn

- **[Quick Start](/using-floww/quick-start/)** - Get up and running in minutes
- **[Core Concepts](/using-floww/concepts/)** - Understand triggers, providers, and workflows
- **[Running Locally](/using-floww/running-locally/)** - Development setup and debugging

## What You Can Build

- **Webhook handlers** - Respond to GitHub, GitLab, or any HTTP webhooks
- **Scheduled tasks** - Run code on cron schedules
- **Event-driven workflows** - React to calendar events, repository changes, and more
- **Multi-service integrations** - Connect different APIs and services seamlessly
