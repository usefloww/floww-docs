# Reference

Complete reference documentation for Floww SDK providers and APIs.

## Providers

Each provider offers triggers and utilities for specific platforms:

- **[Builtin](/reference/builtin/)** - Core webhook and cron triggers (always available)
- **[GitLab](/reference/gitlab/)** - Repository events, merge requests, and issues
- **[Google Calendar](/reference/google_calendar/)** - Calendar events and scheduling automation

## Core API

- **[API Reference](/reference/api/)** - Dynamic code execution, context utilities, and error handling

## Quick Start

1. **Choose a provider** based on what events you want to handle
2. **Import and configure** the provider in your workflow file
3. **Define triggers** that respond to specific events
4. **Use the context API** for logging, storage, and configuration
